Drilling 1

Este ejercicio trata de realizar un cajero automático con ingreso y validación de usuario, consulta de saldo, realizar gira o depósitos.

Requisitos:

  1. Cuenta Bancaria con:
    • Nombre de usuario
    • Identificador
    • Password
    • Saldo en su cuenta
  2. Verificación de nombre de usuario y password
  3. Menú para ver movimientos del usuario.
